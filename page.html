EFFACER LES DONNÉES DE L'APPAREIL
Toutes les données seront supprimées définitivement de cet appareil. Vous ne pourrez plus localiser ce dernier.
Si votre appareil est hors connexion, la suppression commencera lors de sa prochaine connexion.
Pour effacer les données de votre appareil, vous devez peut-être vous reconnecter à votre compte Google.
Pour naviguer, appuyez sur les touches fléchées.
